OpenGuessr 🌍📌

PaulPlay
Tags : Gratuit ; Alternative ; Géographie ; Multijoueur/Solo

OpenGuessr est un jeu basé sur les données StreetView de Google Maps. Très inspiré du jeu GeoGuessr, cette alternative est gratuit et illimité.

Le but du jeu est de, grâce aux images panoramiques, trouver l'emplacement où vous êtes. Il existe des fonctionnalités comme lancer une partie avec uniquement des emplacements dans un pays cible, dans des capitales, dans des endroits connus, ...

GPOP.io 🕹️🎶

Tags : Gratuit ; Rythme ; Musique ; Multijoueur/Solo

GPOP est un jeu de rythme similaire à Guitar Hero ou à Fortnite Festival où les niveaux sont créer par la communauté et qui fonctionne grâce à YouTube.

Le but du jeu est de taper les touches indiquées en rythme sur une musique.

Infinite Craft ♾️🔨

Neal Fun
Tags : Gratuit ; Sandbox ; Créatif ; Solo

Infinite Craft est un jeu totalement gratuit utilisant l'IA pour générer des combinaisons infinies.

Le principe du jeu est de combiner 2 éléments pour en faire un nouvel élément.

Slow Roads 🚘🛣️

Anslo
Tags : Gratuit ; Conduit ; Relax ; Solo

Slow Roads est un jeu de conduite apaisant où vous explorez des paysages générés de manière procédurale. Sans objectif ni limite de temps, profitez simplement de la route et du décor.

Vous pouvez personnaliser votre expérience en changeant le type de terrain, la météo, le temps, le type de véhicule, ...

NZ : Portable 🧟‍♂️

NZ : P Team
Tags : Gratuit ; Zombie ; Demake ; Multijoueur/Solo

NZ : Portable ou plutôt Nazi Zombie : Portable est un demake du mode Zombie de Call of Duty World at War. Cette reproduction est constituer de plusieurs cartes crées par la communauté.

Son but est le même que celui de Call of Duty, tuer des zombies et survivre le plus possible.

Noclip 🎥🗺️

Jasper
Tags : Gratuit ; Visualiseur ; Exploration ; Solo

Noclip est un site qui a pour but de vous donner la possibilité d'explorer des cartes de jeux en caméra noclip (caméra libre) comme Mario Kart, Wii Sport, Dark Souls ou même GTA San Andreas.

AutoGuessr 🚗🔍

PaulPlay
Tags : Gratuit ; Automobile ; Quiz ; Multijoueur/Solo

AutoGuessr est un jeu du même créateur que OpenGuessr. Il est axé sur le thème des voitures.

Son but est simple, grâce aux caractéristiques d'une voiture ainsi que ses photos, retrouver son prix en dollar.

Wolfy 🐺🏘️

Tags : Gratuit ; Stratégie ; RolePlay ; Multijoueur

Wolfy est un jeu inspiré des Loups-Garous de Thiercelieux. Dans une partie, chaque personne à un rôle mais on un objectif à atteindre :

- Pour les Loups-Garous, exterminer tout le village et qu'il reste uniquement eux.

- Pour les autres, exterminer tous les Loups-Garous.

Sur Wolfy, il y a des rôles exclusifs (inexistant dans la version de Thiercelieux) : le Dictateur, le Chaperon rouge, bleu et loup, le Loup bavard, l'Héritier, le Pillars, le Mercenaire, le Mentaliste, le Fossoyeur ainsi que la Voyante d'aura.

Nécessite la création d'un compte pour y jouer

LostGamer 🕹️📌

Tags : Gratuit ; Jeux-Vidéos ; Quiz ; Multijoueur/Solo

LostGamer a pour même principe que GeoGuessr, trouver l'emplacement où vous êtes grâce à des images panoramiques sauf qu'ici, on est pas dans le monde réel mais plutôt dans des cartes de jeux-vidéos comme GTA V, Fortnite, Skyrim ou encore Genshin Impact.
